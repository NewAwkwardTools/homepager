# Dependency

* Python 3.6+
* Python library
    * toml
* [titlegetter](https://github.com/WeepingDogel/TitleGetter)

# Installation

By git:
```
$ git clone https://github.com/NewAwkwardTools/homepager.git
```

By [aur](https://aur.archlinux.org/packages/homepager/)
```
$ yay -S homepager
```
